# Projeto Python_SQL_MongoDB

Este projeto é uma aplicação desenvolvida em Python que integra bancos de dados SQL e MongoDB, demonstrando como utilizar os dois tipos de armazenamento de forma eficiente. O objetivo é permitir que os usuários possam interagir com dados provenientes de ambos os sistemas de forma fluida.

## Estrutura de Diretórios

```
/
├── sql/
│   ├── models.py     # Modelos para a base de dados SQL
│   ├── database.py    # Conexões e operações para SQL
├── mongodb/
│   ├── models.py     # Modelos para a base de dados MongoDB
│   ├── database.py    # Conexões e operações para MongoDB
├── main.py           # Script principal para execução da aplicação
├── requirements.txt   # Dependências do projeto
└── README.md         # Documentação do projeto
```

O projeto é estruturado em diferentes diretórios para separar as funcionalidades relacionadas aos bancos de dados SQL e MongoDB, tornando o código mais organizado e mantendo uma boa arquitetura.
